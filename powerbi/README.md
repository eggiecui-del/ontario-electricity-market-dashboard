# Power BI Files

The Power BI Desktop file is saved here:

```text
ontario_electricity_dashboard.pbix
```

The dashboard was built in Power BI Desktop using the notes in this folder.

Use PostgreSQL as the source and connect to these views:

- `v_hourly_market_summary`
- `v_daily_market_summary`
- `v_monthly_market_summary`
- `v_market_stress_hours`
